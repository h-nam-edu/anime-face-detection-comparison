# Data Directory

This folder is intentionally left empty in the repository to avoid tracking large binary image files. 

When you run the `01_comparative_study.ipynb` notebook, the scripts will automatically query the Kaggle and Hugging Face APIs to download and unpack the necessary datasets into this directory. 

* **Note:** The generated YOLO dataset (`anime_face_yolo_dataset`) will also be built locally and should not be committed to version control.
